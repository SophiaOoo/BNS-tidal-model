# BNS-tidal-model
This repository contains the python files and notebooks used to generate linear perturbation models of binary neutron stars.

The notebook "BNS-tidal-model" is a consolidated version of all my code that runs independantly of any external python files. This notebook also contains an in depth explanation of the math behind the model.

All python files and the notebook "BinaryNS" could be used to recreate earlier iterations of the model. Note that paths in these files would need to be modified.
